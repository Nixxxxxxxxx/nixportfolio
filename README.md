# Nix Portfolio

Foundation for a personal portfolio website built with Next.js, TypeScript,
Motion, and Vercel.

## Stack
- Next.js 16 App Router
- React 19
- TypeScript
- Motion
- Vercel

## Requirements
- Node.js `>= 20.9.0`

## Scripts
- `npm install`
- `npm run dev`
- `npm run lint`
- `npm run typecheck`
- `npm run build`

## Git Workflow
- `main` is the stable production branch
- `codex/*` branches are used for isolated tasks and preview deployments

## Current Status
- Foundation is ready
- The next step is implementing real sections from Figma
